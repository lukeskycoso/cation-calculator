Roadmap for Cation-Calculator
------------------------------------------------------------
Version 1.0.1
COMPLETE - automatic identification of oxides present
COMPLETE - automatic calculation of cations from oxide weight percentage (Ox wt%) to atoms per formula unit (apfu)
COMPLETE - automatic saving the output results in .csv format for maximum compatibility with all operating systems

Version 1.0.2
- add optional calculation of Fe2+/Fe3+ partition with Droop (1987) method
- add optional backward calculation of oxides weight percentage to estimate the FeO and Fe2O3 content values
	+ to this, add the re-recalculation of cations, based on the new recalculated oxides proportions

Version 1.0.3
- add cations site occupation for pyroxenes
- add main classification diagrams for pyroxenes and plagioclases
- add options for secondary classification diagrams (for example QUAD-kushiroite-esseneite diagram for pyroxenes)

Version 1.0.4
- add optional Nimis and Taylor (2000) thermobarometry for cpx coming from garnet lherzolites, with the inclusion of data filtering for Ziberna et al.(2016).
- add option for qualitative data filtering

Version 1.0.5
- add cations site occupation for garnet
- add main classification diagrams for garnet

Version 1.0.6
- add main classification diagrams for spinel and olivine
- add optional olivine-ilmenite thermometry of Andersen and Lindsey (1979)
